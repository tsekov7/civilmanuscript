# Civil Manuscript / Граждански рѫкописъ
Civil Manuscript is a historical Cyrillic typeface based on 19th-century Bulgarian handwriting and orthography. Inspired by the 1879 "Petit manuel pour apprendre la langue Bulgare" by P. V. Gorbanoff (published in Sofia), this font preserves extended Cyrillic glyphs including ѣ, ѫ, ѭ. 
Designed for historical reconstructions, educational materials, and cultural heritage projects.
